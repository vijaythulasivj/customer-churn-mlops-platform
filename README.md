# Customer Churn MLOps Platform

## Project Overview

This project demonstrates an end-to-end MLOps pipeline using:

- Apache Airflow
- Kubeflow
- MLflow
- Jenkins
- Docker
- Kubernetes
- FastAPI
- Terraform
- PostgreSQL

## Project Workflow

GitHub
→ Jenkins
→ Docker
→ Kubernetes
→ Airflow
→ Kubeflow
→ MLflow
→ FastAPI
→ Monitoring
